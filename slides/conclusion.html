Выводы
  • KnockoutJS предназначен для быстрой разработки сложного UI
  • MVVM штука не привычная, сложно перестроится после MVC
  • Может быть внедрён на любой стадии проекта
  • Совместим с любым DOM Manipulation framework'ом
  • Для сложных приложений нужно писать свой фундамент или использовать Backbone
  • На выходе получаем много reusable кода
