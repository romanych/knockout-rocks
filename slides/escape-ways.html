Пути выхода из кризиса

jQuery/MooTools/Prototype

  • Позволяют забыть о зоопарке браузеров
  • Расширяют возможности браузеров
  • Не помогают решить проблему лапшекода

MVC

  • Наверное старейший архитектурный шаблон
  • Множество реализаций
  • Подходит для Stateless и Stateful приложений

MVVM

  • Молодой
  • Мало реализаций
  • Заточен для Stateful приложений
